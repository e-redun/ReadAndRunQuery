# ReadAndRunQuery
 Test, Sql Server 2005, stored procedure, хранимые процедуры

Тестовое задание: Написать консольное приложение. 
Приложение по команде считывает из файла input.txt sql запрос, выполняет его и записывает результат выполнения в файл output.csv. Возникающие ошибки выводятся на экран консоли и не завершают выполнение программы. Предусмотреть возможность подключения к базе данных Microsoft SQL Server через конфиг.
